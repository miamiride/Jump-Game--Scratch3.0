Jump game (Scratch 3 Project)



## How to Play
1. Start the game by clicking the green flag.
2. Jump to prevent hitting obstacles that moves across the screen.
3. Each time you jump, your score increases and speed.

## Features
- Score counter
- Beginner-friendly coding logic
- Teaches basic Scratch 3 blocks: motion, events, loops, and conditions

## Learning Outcomes
- Understanding of sprite movement
- Basic event handling in Scratch
- Introduction to game logic and collision detection

## How to Use
1. Open [Scratch 3](https://scratch.mit.edu/) editor.
2. Create a new project or open the `.sb3` file provided.
3. Explore and modify the sprites, scripts, and backgrounds.
4. Experiment with adding your own features.
